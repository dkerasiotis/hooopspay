# Changelog

## [2.1.0] - 2026-03-20

### Προστέθηκε
- Ο admin μπορεί να αναθέτει δραστηριότητες σε χρήστες κατά τη δημιουργία (dropdown "Ανάθεση σε χρήστη")
- Ο admin μπορεί να αλλάζει τον κάτοχο μιας υπάρχουσας δραστηριότητας κατά την επεξεργασία

---

## [2.0.0] - 2026-03-20

### Προστέθηκε
- Multi-user authentication (bcrypt hashed passwords, session-based)
- Ρόλοι χρηστών: admin (βλέπει όλα) και user (βλέπει μόνο τα δικά του)
- Login page
- Αλλαγή κωδικού από τον ίδιο τον χρήστη
- Διαχείριση χρηστών από admin (δημιουργία, reset κωδικού, διαγραφή)
- Favicon (SVG)
- Health endpoint (`/api/health`)

### Αλλαγή
- Μετονομασία "Ομάδες" → "Δραστηριότητες" σε όλη την εφαρμογή
- Port εφαρμογής: 3000 → 5002
- Credentials αποθηκεύονται στη βάση αντί για env vars
- Generic branding (δεν αφορά μόνο μπάσκετ)

---
